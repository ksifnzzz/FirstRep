# CursorProject

Python 프로젝트 템플릿입니다. (환경: Windows, PowerShell)

## 설정

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## 실행

```powershell
python -m src.main
```

## 테스트

```powershell
pip install pytest
pytest tests/
```
